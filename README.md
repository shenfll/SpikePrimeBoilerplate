### Tank Movement

<h3>Gyro Straight for Rotations</h3>  

```python
mt(amount, ls, rs, wait = True)
```
Drives straight using the gyro on Port.S2  
<b>Parameters:</b>  
>amount => Distance in degrees
>ls => Left motor speed
>rs => Right motor speed
>wait => Run motor asynchronously?
